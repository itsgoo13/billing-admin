🌐 R31NET INTERNET SERVICE
Jl Otista Gg H Maung Rt 09/018 No 31, Kedaung - Pamulang
☎ ${telepon || '0878-8986-3799'}
Invoice #: ${invoiceNumber}
Tanggal: ${tglStr}
Pelanggan: ${nama}
IP: ${ip}
${instalasiNum > 0 ? `` : ''} ${sewaNum > 0 ? `` : ''} ${diskonNum > 0 ? `` : ''}
DeskripsiJumlah
Paket ${paket}Rp ${hargaNum.toLocaleString('id')}
Biaya InstalasiRp ${instalasiNum.toLocaleString('id')}
Sewa AlatRp ${sewaNum.toLocaleString('id')}
Diskon- Rp ${diskonNum.toLocaleString('id')}
TOTAL: Rp ${total.toLocaleString('id')}
${keterangan ? `
📝 Catatan: ${keterangan}
` : ''}
💳 Metode Pembayaran:

Transfer Bank:
BCA: 7310685440 a.n VIA DESIANA

Scan QRIS:

Scan QR di atas menggunakan aplikasi pembayaran (DANA/OVO/GoPay/BCA Mobile)

DANA/OVO: 087898863799

Terima kasih atas kepercayaan Anda!

`; const blob = Utilities.newBlob(html, 'text/html'); const pdf = blob.getAs('application/pdf').setName('Invoice_' + nama + '.pdf'); let fileUrl = ''; try { const folder = DriveApp.getFolderById('YOUR_FOLDER_ID'); // Ganti jika perlu const file = folder.createFile(pdf); fileUrl = file.getUrl(); } catch (e) { const tempFile = DriveApp.createFile(pdf); fileUrl = tempFile.getUrl(); } return { success: true, url: fileUrl }; }
